### Overview

General idea is to figure out a fun topic to build a demo project.
Uses https://crates.io/crates/crates_io_api API.

#### Usage
cd into `rust-ideas`

`cargo run -- --help`
`cargo run -- popular --number 4`
`cargo run -- random`
